## BACKEND

### 1. Deteccao Urbano/Rodovia (PostGIS + roadway_info)

A tabela `roadway_info` ja existe no schema (Task 01). Implementar a logica:

- Query PostGIS: verificar se a localizacao do cliente esta proxima de uma rodovia cadastrada
- Se distancia < 500m de rodovia: contexto = "highway"
- Se nao: contexto = "urban"
- Impacta: multiplicador de preco (highway x1.15, rural x1.30) e raio de busca (highway 15-30km)

**Endpoint**: Integrar na criacao do service-request (automatico, nao input do usuario)

**Seed**: Adicionar rodovias principais de SP no script de seed (Bandeirantes, Anhanguera, Dutra, Castello Branco)
